¿Qué es Docker?
Docker es una plataforma que permite crear, ejecutar y administrar aplicaciones en contenedores. 
Sirve para que un programa funcione igual en cualquier computadora, sin importar el sistema operativo.

¿Qué es el Dockerfile?
Es un archivo de texto con instrucciones que le dicen a Docker cómo construir una imagen. 
Por ejemplo, qué sistema operativo usar, qué programas instalar, y qué comando ejecutar.

¿Qué es Docker Image?
Es como una foto o plantilla que incluye todo lo necesario para ejecutar un programa (código, librerías, dependencias). 
Se genera a partir del Dockerfile.

¿Qué es un Docker Container?
Es una instancia en ejecución de una imagen. 
Un contenedor es como “levantar” la aplicación lista para usarse.

Comparación:
- Dockerfile: receta (instrucciones).
- Docker Image: pastel ya preparado (plantilla).
- Docker Container: cuando te comes el pastel (ejecutar la app).
